Windows Notes
=============

BitBlocks Core Windows binaries are published at:

https://github.com/BitBlocksProject/BitBlocks/releases

Common binaries:

- bitblocks-qt.exe: graphical wallet
- bitblocksd.exe: daemon
- bitblocks-cli.exe: command-line RPC client
- bitblocks-tx.exe: transaction utility built from source; it may not be included in every installer package

The default Windows data directory is:

%APPDATA%\BitBlocks

Mainnet defaults:

- P2P port: 58697
- RPC port: 59768

To build Windows binaries from source, see build-windows.md.
